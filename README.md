# Writing Repository

This repository is for storing various writing documents, notes, and articles.

## Purpose

- Store writing projects
- Organize notes and ideas
- Version control for documents
- Collaborative writing

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/chen01234/writing.git
   ```

2. Create a new document in the appropriate folder.

3. Commit and push your changes.

## Document Structure

- `/articles/` - Long-form articles and essays
- `/notes/` - Quick notes and ideas
- `/drafts/` - Work in progress
- `/published/` - Completed documents
